# 野钓App MVP 部署文档

## 环境要求

- Docker 20.10+
- Docker Compose 2.0+
- Node.js 20+ (开发环境)

## 快速启动 (Docker Compose)

### 1. 创建 docker-compose.yml

```yaml
version: '3.8'

services:
  postgres:
    image: postgres:15-alpine
    container_name: yediao-postgres
    environment:
      POSTGRES_USER: yediao
      POSTGRES_PASSWORD: yediao123
      POSTGRES_DB: yediao
    ports:
      - "5432:5432"
    volumes:
      - postgres_data:/var/lib/postgresql/data
      - ./database/init.sql:/docker-entrypoint-initdb.d/init.sql
    healthcheck:
      test: ["CMD-SHELL", "pg_isready -U yediao"]
      interval: 5s
      timeout: 5s
      retries: 5

  redis:
    image: redis:7-alpine
    container_name: yediao-redis
    ports:
      - "6379:6379"
    volumes:
      - redis_data:/data

  backend:
    build: ./backend
    container_name: yediao-backend
    environment:
      NODE_ENV: production
      PORT: 3000
      DB_HOST: postgres
      DB_PORT: 5432
      DB_USER: yediao
      DB_PASSWORD: yediao123
      DB_NAME: yediao
      JWT_SECRET: yediao-secret-key-2026-mvp
      REDIS_HOST: redis
      REDIS_PORT: 6379
    ports:
      - "3000:3000"
    depends_on:
      postgres:
        condition: service_healthy
      redis:
        condition: service_started

  web:
    build: ./web
    container_name: yediao-web
    ports:
      - "3001:3000"
    environment:
      NEXT_PUBLIC_API_URL: http://localhost:3000/api/v1
    depends_on:
      - backend

volumes:
  postgres_data:
  redis_data:
```

### 2. 启动服务

```bash
cd yediao-app
docker-compose up -d
```

### 3. 验证服务

- 后端API: http://localhost:3000/api/v1
- API文档: http://localhost:3000/api/docs
- Web前端: http://localhost:3001

## 手动部署

### 后端部署

```bash

npm run build
npm start
```

### Web前端部署

```bash
cd web
npm install
npm run build
npm start
```

```bash
psql -U yediao -d yediao -f database/init.sql
```

## 生产环境建议

1. 使用 Nginx 反向代理
2. 配置 HTTPS (Let's Encrypt)
3. 启用 PostgreSQL 主从备份
4. 配置 Redis 持久化
5. 使用 PM2 管理 Node.js 进程
6. 配置日志收集 (ELK/Loki)

## 常见问题

### 1. 数据库连接失败
检查 PostgreSQL 服务是否启动，环境变量是否正确。

### 2. 验证码收不到
MVP阶段使用固定验证码 `123456`，实际生产环境需接入短信服务商。

### 3. 地图不显示
需要配置高德地图 API Key，在 Web/Mobile 代码中替换。

### 4. AI识鱼失败
MVP阶段使用模拟数据，实际生产环境需配置百度EasyDL API Key。
