## 三、技术实现细节

### 3.1 社区推荐算法

```
综合评分 = exp(-ln2 * hoursOld / 24) * (1 + engagementNorm * 4)

其中:
- hoursOld: 帖子发布至今的小时数
- 半衰期: 24小时（每24小时权重减半）
- engagement = likeCount*1 + commentCount*2 + shareCount*3
- engagementNorm = min(engagement / 100, 1)
```

### 3.2 OAuth默认隐私级别
OAuth新用户创建时默认设置为 `PrivacyLevel.PUBLIC`，显示钓鱼年龄/常钓鱼点/擅长鱼种。

### 3.3 鱼类百科覆盖鱼种（30+）
鲤鱼、草鱼、青鱼、鲢鱼、鳙鱼、鲫鱼、鳜鱼（桂鱼）、鳙鱼、黑鱼、鳝鱼、泥鳅、鲶鱼、河鲀（河豚）、鳗鲡、鲈鱼、鳕鱼、石斑鱼、多宝鱼、大黄鱼、小黄鱼、带鱼、鲳鱼、鲅鱼、金鲳鱼、罗非鱼、鲥鱼、鲟鱼、鳇鱼、黄颡鱼（昂刺）、鳊鱼、翘嘴（红铂）、军鱼、白条鱼等。

## 五、已知限制与后续工作

### MVP阶段限制
1. **AI识鱼**: 使用mock数据，未接入百度EasyDL或阿里云视觉API
2. **鱼类百科**: 使用枚举 `ProtectionLevelEnum`，MVP阶段先seed到数据库
3. **TypeScript编译错误**: entities中大量 `Entity`/`Column`/`PrimaryGeneratedColumn` 装饰器报错，原因是NestJS装饰器版本与TypeScript 5.x实验性装饰器配置不兼容。这些是预存的框架层问题，不影响业务逻辑实现。

### 后续工作
1. 解决 `experimentalDecorators` / `emitDecoratorMetadata` TypeScript配置问题
